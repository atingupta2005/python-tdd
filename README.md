# Python TDD
### How to run tests locally
```
git clone https://github.com/atingupta2005/python-tdd
cd python-tdd
ls
```

```
python3 -m venv venv
```

```
./venv/bin/activate
pip install -r requirements.txt
pip install -U pip setuptools
pip install pytest
pip install pytest-cov
```


```
pytest
pytest --cov=primes
```
